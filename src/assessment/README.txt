Task 1:
Simply run the code and enter an amount of money in pence. The console will return the output of the coins received in change.

Task 2:
Change the filepath of the variable "filepath" to match the coin-inventory.properties file on your machine. 
Use // within the filepath where your machine would indicate / e.g. C:/emilyscomputer/coin-inventory.properties should be C://emilyscomputer//coin-inventory.properties
Run the code.
The code will read in the file and if there is no money in the vending machine the code will exit.
If there are coins, enter an amount of money in pence.
If the machine has run out of coins to give, it will tell you the amount of pence you are missing.
The console will return the output of the coins received in change.
